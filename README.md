# Fly + Go + Fiber

Repository for
the [Deploying a Go Fiber application to Fly](https://technotrampoline.com/articles/deploying-a-go-fiber-application-to-fly/) article.

## Steps

```bash
go mod init hello-world
```

```bash
go get github.com/gofiber/fiber/v2
```

To run the application locally:

```bash
go run ./src/application.go
```
